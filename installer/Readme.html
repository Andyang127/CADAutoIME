🎯 产品概述

🔹 产品定位

CAD Auto IME 是一款专为工程设计领域打造的底层系统级效率增强组件。v0.3 版 融合了双擎状态机锁与底层焦点雷达，彻底消灭了绘图卡顿与输入法误杀，真正实现了 0 延迟的无感输入法切换体验。

🔹 解决的核心问题

使用场景 传统操作方式 (含旧版插件) 本工具解决方案 (v0.3 版)
日常绘图 (快捷菜单弹出) 极易因为弹窗抢走焦点被误切成中文 ✅ 状态锁+黑名单雷达，画线(PL)等弹窗时死锁英文，绝不误判
各种文本与图块标注 经常因层级问题导致输入法无法呼出 ✅ Win32 底层 API 穿透嵌套界面，精准捕获打字光标瞬间切中文
多图纸切换 (MDI) 图纸 A 的中文状态会污染图纸 B ✅ 独立生命周期隔离，切回图纸瞬间自动“洗牌”重置为英文

⚡ v0.3 版核心特性

本次版本迎来了史诗级架构跃迁，让插件兼具稳健与智能：

1. 🧠 双擎状态锁：0 误杀防御网

我们将“命令状态”与“焦点弹窗”深度结合。只要你执行的不是白名单里的文字命令，底层就会挂上一把“死锁”，即使界面弹出一万个下拉菜单抢占焦点，插件也绝对不会切出中文。而在使用天正、源泉等图块重命名插件时，则秒切中文。

2. 🖥️ 全透明双区平权 UX 与无感交互

彻底抛弃“黑盒”逻辑。现在的配置面板中，所有 AutoCAD 核心文字命令（如 TEXT, MTEXT, BLOCK, LAYER 等 28 个）与外挂命令完全透明可见。搭配极简的“双击删除”与底部 Toast 无感文字提示，交互体验如丝般顺滑。

3. 💊 零配置自愈升级机制

插件内置了智能向下兼容探针。如果你是从老版本升级上来的用户，程序在后台启动时会瞬间发现你的配置文件缺失了新功能指令，并以纳秒级速度静默为你修补完整。你什么都不用做，即刻享受满血体验。

4. 🚀 跨世代架构引擎 (兼容 CAD2007~2027)

底层核心经过全面重构，向下兼容至 .NET 3.5 时代的老爷机，向上完美适配未来采用 .NET 10.0 的 AutoCAD 2027，真正做到“全宇宙版本通杀”。

📖 使用指南与命令

💡 极简操作提示：经过大量测试，本版的本程序自带97个内置命令，包括 46 个 CAD 原生系统命令和 51 个常用插件命令。你可以直接上手画图，如果发现有极个别自定 LISP 没有切中文，再使用下方命令添加即可。

⚠️ 添加自定义命令的铁律：必须输入“命令全称”！
当你在配置面板中添加自己的扩展命令时，严禁输入快捷键（Alias）（例如画圆不要输 C，创建块不要输 B）。
你必须输入 CAD 底部命令行执行时的真实全称（例如：CIRCLE、BLOCK、TBLKNAME 等），否则系统底层将无法匹配触发！

🔹 1. 呼出白名单配置面板

在 CAD 命令行输入：👉 CUSTOMAUTOIME

功能：直观管理所有文字、标注指令。支持 Ctrl+Z 撤销，Ctrl+Y 重做，以及双击/Delete 一键防呆删除。

🔹 2. 临时开启/关闭助手

在 CAD 命令行输入：👉 TOGGLEAUTOIME

功能：一键全局切换插件的运行状态（开启/关闭监控）。

❓ 常见问题

Q1：我可以把面板下方的 TEXT、MTEXT 等系统内置指令删掉吗？

A：强烈建议保留！ 它们是维持 AutoCAD 正常文字录入的核心骨干。虽然我们将控制权完全交给了你（允许你强行删除），但删除后，这些命令将不再自动切换输入法。如果误删，请点击面板下方的【恢复默认】按钮一键重置。

Q2：程序会影响 AutoCAD 运行速度、引起掉帧卡顿吗？

A：绝对不会。本次 v0.3 版本彻底消灭了高频拦截带来的垃圾回收（GC）压力，通过底层内存 Buffer 复用机制，对 CPU 和帧率的影响严格为 0。哪怕连续画图 10 小时，鼠标依然顺滑如初。
