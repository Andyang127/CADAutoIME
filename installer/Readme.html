v0.3.1 版核心特性

本次版本在 v0.3 史诗级架构基础上，聚焦稳定性与工程化打磨，让插件更稳、更可靠：

1. 双擎状态锁：0 误杀防御网

我们将"命令状态"与"焦点弹窗"深度结合。只要你执行的不是白名单里的文字命令，底层就会挂上一把"死锁"，即使界面弹出一万个下拉菜单抢占焦点，插件也绝对不会切出中文。而在使用天正、源泉等图块重命名插件时，则秒切中文。

2. 多文档状态物理隔离（v0.3.1 新增）

彻底解决多图纸作业的状态串扰问题。每张图纸拥有独立的输入法状态机，切换文档时自动重置，再也不会出现"图纸 A 打完字，切到图纸 B 画线时莫名其妙出中文"的尴尬。

3. 线程安全加固（v0.3.1 新增）

底层焦点雷达全面升级并发安全防护。重构缓冲区管理策略，从根源上杜绝多线程并发导致的潜在风险，长时间运行更稳定。

4. 全透明双区平权 UX 与无感交互

彻底抛弃"黑盒"逻辑。现在的配置面板中，所有 AutoCAD 核心文字命令（如 TEXT, MTEXT, BLOCK, LAYER 等 28 个）与外挂命令完全透明可见。搭配极简的"双击删除"与底部 Toast 无感文字提示，交互体验如丝般顺滑。

5. 零配置自愈升级机制

插件内置了智能向下兼容探针。如果你是从老版本升级上来的用户，程序在后台启动时会瞬间发现你的配置文件缺失了新功能指令，并以纳秒级速度静默为你修补完整。你什么都不用做，即刻享受满血体验。

6. 跨世代架构引擎 (兼容 CAD2007~2027)

底层核心经过全面重构，向下兼容至 .NET 3.5 时代的老爷机，向上完美适配未来采用 .NET 10.0 的 AutoCAD 2027，真正做到"全宇宙版本通杀"。

使用指南与命令

极简操作提示：经过大量测试，本程序的当前版本自带 97 个 内置命令，包括 46 个 CAD 原生系统命令和 51 个 常用插件命令。可以直接上手画图，如发现有个别自定 LISP 没有切中文，再手动添加即可。
添加自定义命令的铁律：必须输入“命令全称”！
在配置面板中添加扩展命令时，严禁输入快捷键（Alias）（例如画圆不要输 C，创建块不要输 B）。你必须输入 CAD 底部命令行执行时的真实全称（例如：CIRCLE、BLOCK、TBLKNAME 等），否则系统底层将无法匹配触发！

🔹 1. 呼出白名单配置面板

在 CAD 命令行输入该命令，即可直观管理所有文字、标注指令。支持 Ctrl+Z 撤销，Ctrl+Y 重做，以及双击/Delete 一键防呆删除。

COMMAND: CUSTOMAUTOIME

🔹 2. 临时开启 / 关闭助手

在 CAD 命令行输入该命令，即可一键全局切换插件的运行状态（开启/关闭监控）。

COMMAND: TOGGLEAUTOIME

常见问题解答

Q1：我可以把面板下方的 TEXT、MTEXT 等系统内置指令删掉吗？

A：强烈建议保留！它们是维持 AutoCAD 正常文字录入的核心骨干。虽然我们将控制权完全交给了你（允许你强行删除），但删除后，这些命令将不再自动切换输入法。如果误删，请点击面板下方的【恢复默认】按钮一键重置。

Q2：程序会影响 AutoCAD 运行速度、引起掉帧卡顿吗？

A：绝对不会。v0.3 版本彻底消灭了高频拦截带来的垃圾回收（GC）压力，通过底层内存 Buffer 复用机制，对 CPU 和帧率的影响严格为 0。v0.3.1 进一步优化了 HUD 定时器频率，哪怕连续画图 10 小时，鼠标依然顺滑如初。

Q3：同时打开多张图纸会互相影响吗？

A：v0.3.1 版本已彻底解决此问题。每张图纸拥有独立的输入法状态机，完全物理隔离。在图纸 A 中输入文字切到中文，切到图纸 B 画线时自动恢复英文，互不干扰。

卸载流程

本程序提供极其纯净的卸载体验，不留任何系统垃圾。

  1. 打开 Windows 「开始菜单」，找到 「CAD Auto Ime」 程序组。
  2. 点击 「卸载 CAD 输入法助手」 选项。
  3. 注意：为确保彻底清除注册表占用，卸载时若 CAD 正在运行，程序会弹出友好提示建议您保存图纸并协助自动关闭 CAD 进程。
